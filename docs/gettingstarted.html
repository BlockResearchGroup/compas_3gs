Installation

The recommended way to install the main library of COMPAS is to use Anaconda/conda::

$ conda config --add channels conda-forge
$ conda install COMPAS

But it can also be installed using pip::

$ pip install COMPAS

After the installation, start Python from the command line and verify your setup::

>>> import compas
>>> import compas_rhino
>>> import compas_blender
>>> import compas_ghpython

Optionally, you can also install from source.
