CellNetwork.edge_vector

CellNetwork.edge_vector(u, v, unitized=True)

Return the vector of an edge.

Parameters

  • u (hashable) – The key of the start node.

  • v (hashable) – The key of the end node.

Returns

list[float] – The vector from u to v.
